# GESTION DE CAFETERIA 🔨📦

**Git y Github**

Una aplicacion que sirve para llevar el registro de unos cafes en una cafeteria
---

## 🧩 Problemática que resuelve


- Registrar movimientos de los cafes.
- Validar operaciones según disponibilidad y estado.
- Evitar inconsistencias en stock ante errores o anulaciones.
